To publish:
1. Replace placeholder images with your real files while keeping the same filenames.
2. Upload the full folder contents to any static host such as GitHub Pages, Cloudflare Pages, or Netlify Drop.
3. Make sure all files stay in the same folder level.

To change the featured video later:
- Replace VXPDXHJLjWQ in index.html and digital-library.html with the new YouTube video ID.
